# Mobile UX Rules

- Mobile-first layouts: full-width inputs, labels above fields.
- Thumb ergonomics: navigation at bottom, primary action buttons within thumb reach.
- Touch target sizing: buttons, links, and click target area at least 44x44px.
- Use Bottom Sheets instead of deep nested modals for complex sub-workflows.
- Dynamic safe area handling on iOS and Android viewports.
